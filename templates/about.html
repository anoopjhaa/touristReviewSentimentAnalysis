Tourist Place Reviews Sentiment Analysis

using Machine Learning Techniques

Project Description

Millions of user review and rate tourist places as Social media is growing trend now a days. Sentiment analysis can be performed over these reviews to determine popular place. Based on sentiment analysis result, tourist can easily decide the destination to be visited.

The Dataset will be collected from various tourism review websites using data scraping techniques. A Comparative study will be performed on the various sentiment analysis techniques to determine the best technique for the data.
